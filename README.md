- “你好吗？”

### 第二课：
- 学习内容：“你叫什么名字”，“很高兴认识你”，“你第一次来上海吗”，“我是志愿者”，“你家在哪”，“我家在新疆乌鲁木齐”，“你几岁啦？ 二十六。”，“你家有几口人？”，“五口，爸爸，妈妈，姐姐，哥哥，和我”
